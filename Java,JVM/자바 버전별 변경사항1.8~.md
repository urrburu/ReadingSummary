## 1.10

형을 별도로 지정하지 않는 var 변수를 사용할 수 있게 되었다. 

String 영역에서 큰 변화가 있었는데 공백문자 포함여부를 확인하는 함수 isBlank(), lines(),  repeat() 처럼 String을 라인별로 쪼개거나, 생성하는 옵션들이 많이 생겼다. 
String을 잘라내는 함수들도 많이 생겼다.
